# Shop Plan — One-Shot Purchase Planning

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace per-item LLM calls in shop with a single "shop plan" LLM call that outputs a complete ordered purchase list, then execute mechanically — reducing 3-5 strategic LLM calls to 1.

**Architecture:** New `SHOP_PLAN_TOOL` schema tells the LLM to output an ordered list of purchases with gold tracking. The agent loop stores a `ShopPlan` on `self`, executes items one by one mechanically, and resumes after triggered follow-up scenes (card_reward, card_select, etc.). If gold doesn't match expectations at execution time, the plan is discarded and a new single LLM call is made to replan with the updated state.

**Tech Stack:** Python (Pydantic-free dataclasses, async agent loop, existing V2Engine `decide_noncombat`)

from dataclasses import dataclass, field


@dataclass
class ShopPlanItem:
    """One planned purchase in a shop visit."""
    action: str       # "buy_card" | "buy_relic" | "buy_potion" | "remove_card"
    item_name: str    # Exact name to match against shop state
    price: int
    gold_after: int   # Expected gold after purchase
    reason: str = ""


@dataclass
class ShopPlan:
    """Full purchase plan for a shop visit."""
    items: list[ShopPlanItem] = field(default_factory=list)
    current_index: int = 0          # Next item to execute
    reasoning: str = ""
    strategic_note: str = ""

    @property
    def is_complete(self) -> bool:
        return self.current_index >= len(self.items)

    @property
    def current_item(self) -> ShopPlanItem | None:
        if self.is_complete:
            return None
        return self.items[self.current_index]

    def advance(self) -> None:
        self.current_index += 1
```

This is the core change. The shop flow becomes:

1. Enter shop → auto-open inventory (existing)
2. If no `_shop_plan` → call LLM via `decide_noncombat` to get a plan → parse into `ShopPlan`

    def _parse_shop_plan(self, decision_input: dict) -> ShopPlan | None:
        """Parse a shop_plan decision dict into a ShopPlan object."""
        purchases = decision_input.get("purchases")
        if not isinstance(purchases, list):
            logger.warning("shop_plan: 'purchases' is not a list")
            return None

        items: list[ShopPlanItem] = []
        for p in purchases:
            if not isinstance(p, dict):
                continue
            action = p.get("action", "")
            item_name = p.get("item_name", "")
            price = p.get("price", 0)
            gold_after = p.get("gold_after", 0)
            reason = p.get("reason", "")
            if not action or not item_name:
                continue
            # Normalize action name
            if action == "remove_card":
                action = "remove_card_at_shop"
            items.append(ShopPlanItem(
                action=action,
                item_name=item_name,
                price=int(price) if price else 0,
                gold_after=int(gold_after) if gold_after else 0,
                reason=reason,
            ))

        return ShopPlan(
            items=items,
            reasoning=decision_input.get("reasoning", ""),
            strategic_note=decision_input.get("strategic_note", ""),
        )
```

- [ ] **Step 2: Add `_resolve_shop_item_index()` helper**

```python
    def _resolve_shop_item_index(
        self, gs: GameState, plan_item: ShopPlanItem,
    ) -> int | None:
        """Find the option_index for a planned purchase in the current shop state."""
        shop = gs.shop
        if not shop:
            return None

        name_lower = plan_item.item_name.lower().strip()

        if plan_item.action == "remove_card_at_shop":
            # Card removal doesn't need an index
            return -1  # Sentinel: no option_index needed

        item_lists = {
            "buy_card": shop.cards,
            "buy_relic": shop.relics,
            "buy_potion": shop.potions,
        }
        items = item_lists.get(plan_item.action, [])
        for item in items:
            if not item.is_stocked:
                continue
            item_name = (item.name or "").lower().strip()
            if item_name == name_lower:
                return item.index
        return None
```

- [ ] **Step 3: Add `_execute_shop_plan_step()` method**

This executes one step of the shop plan mechanically:

```python
    async def _execute_shop_plan_step(self, gs: GameState) -> Decision | None:
        """Execute the next item in the active shop plan.

        Returns a Decision on success, or None if the plan should be
        discarded (gold mismatch, item not found, etc.).
        """
        plan = self._shop_plan
        if plan is None or plan.is_complete:
            return None

        item = plan.current_item
        if item is None:
            return None

        # Gold check: if actual gold < item price, discard plan for replan
        if item.action != "remove_card_at_shop":
            if gs.gold < item.price:
                logger.warning(
                    "Shop plan gold mismatch: need %dg for '%s' but have %dg — replanning",
                    item.price, item.item_name, gs.gold,
                )
                self._shop_plan = None
                return None  # Trigger replan

        # Resolve option_index by name
        idx = self._resolve_shop_item_index(gs, item)
        if idx is None:
            logger.warning(
                "Shop plan: item '%s' not found in shop — replanning",
                item.item_name,
            )
            self._shop_plan = None
            return None  # Trigger replan

        # Build and execute the action
        if item.action == "remove_card_at_shop":
            action = {"action": "remove_card_at_shop"}
        else:
            action = {"action": item.action, "option_index": idx}

        logger.info(
            "Shop plan [%d/%d]: %s '%s' (%dg) — %s",
            plan.current_index + 1, len(plan.items),
            item.action, item.item_name, item.price, item.reason,
        )

        await self._execute(action)

        # Post-purchase handling (same as existing _execute_llm_decision)
        if item.action == "buy_relic":
            await self._wait_for_post_shop_relic_transition(gs, idx if idx != -1 else None)
        elif item.action in ("buy_card", "buy_potion"):
            await asyncio.sleep(0.3)

        plan.advance()

        # Gold verification after purchase
        # Re-poll state to check gold
        try:
            raw = await self._client.get_state()
            fresh_gs = parse_state(raw)
            if not plan.is_complete and fresh_gs.state_type != "shop":
                # A follow-up scene was triggered (card_reward, card_select, etc.)
                # The plan stays active — we'll resume when we return to shop
                logger.info(
                    "Shop plan: follow-up scene triggered (%s), will resume after",
                    fresh_gs.state_type,
                )
            elif not plan.is_complete and fresh_gs.gold != item.gold_after:
                # Gold doesn't match expected — might be a relic effect or something.
                # Only replan if we can't afford next item.
                next_item = plan.current_item
                if next_item and fresh_gs.gold < next_item.price:
                    logger.warning(
                        "Shop plan: gold mismatch after '%s' (expected %d, got %d) "
                        "and can't afford next item '%s' (%dg) — replanning",
                        item.item_name, item.gold_after, fresh_gs.gold,
                        next_item.item_name, next_item.price,
                    )
                    self._shop_plan = None
                    # Don't return None here — this step succeeded, return the decision.
                    # Replan will happen on next loop iteration.
        except Exception:
            pass  # Best-effort verification

        return Decision(
            floor=gs.run.floor if gs.run else 0,
            state_type=gs.state_type,
            action=action,
            reasoning=f"Shop plan [{plan.current_index}/{len(plan.items)}]: {item.reason}",
            source="plan",
        )
```

4. **Item not found:** If shop changes unexpectedly → plan discarded → replan
5. **Replan with updated state:** New LLM call sees current gold and remaining items

## Summary of Changes

| Component | Before | After |
|-----------|--------|-------|
| Shop prompt | `build_shop_prompt()` — "choose ONE action" | `build_shop_plan_prompt()` — "plan ALL purchases" |
| Tool schema | `SHOP_TOOL` — single action enum | `SHOP_PLAN_TOOL` — purchases array + skipped_items |
| Decision flow | LLM call per purchase (3-5 calls) | 1 LLM call → mechanical execution |
| Execution | `_execute_llm_decision` per action | `_execute_shop_plan_step` per item |
| Error recovery | Retry single action | Discard plan → replan with fresh state |
| Gold tracking | Implicit (LLM re-evaluates each time) | Explicit (gold_after per step, mismatch → replan) |
