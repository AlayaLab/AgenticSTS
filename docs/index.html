The bounded memory contract

Most long-horizon agents append everything — observations, tool calls, reflections — to every prompt. AgenticSTS implements the opposite contract: every decision is a fresh user message composed by typed retrieval from five knowledge layers, u_d = π(L1, L2(s), L3(s), L4(s), L5(s)), and no raw cross-decision transcript is ever appended. The prompt stays bounded across runs of any length, and any single layer can be ablated in isolation. Full details, reproduction scripts, and 298 condition-tagged trajectories in the repository.
